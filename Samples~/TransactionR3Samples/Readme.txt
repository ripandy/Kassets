Kassets Transaction (R3 Integration) Samples.

In this sample, the usage of Transaction using R3 is demonstrated.
This sample requires R3 to be imported in the project.

TransactionR3.scene

1. Press "Request" button to process a request.
2. Response-side will process the request and return time required to process the request.
3. Response value returned to the request-side, then displayed on the screen.
4. The button will also be disabled until the response is completed.
